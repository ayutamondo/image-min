# HEIC / JPG / PNG 画像圧縮ツール

画像の圧縮・リサイズが行えるツールです。

## 🚀 インストール

```bash
npm install
```

## ▶️ 実行コマンド一覧

| コマンド          | リサイズ（最大 1200px） | JPEG 圧縮（品質 80%） | 説明                                                 |
| ----------------- | ----------------------- | --------------------- | ---------------------------------------------------- |
| `npm run min`     | ○                       | ×                     | 画像を最大 1200px にリサイズのみ行う。圧縮はしない。 |
| `npm run comp`    | ×                       | ○                     | JPEG 画像を品質 80%で圧縮する。サイズ変更はしない。  |
| `npm run mincomp` | ○                       | ○                     | リサイズ（1200px）と圧縮（80%）の両方を実施する。    |

※ HEIC は JPG に変換され、その後同様に処理されます
